# bp.markets.ingest

This repository implements the ingestion pipeline for marketdata

## Dukascopy

[CLICK HERE](./dukascopy)

Python toolkit that downloads, transforms, and resamples Dukascopy’s accurate 1-minute forex data into fast, appendable CSVs.

→ 520+ instrument-years in ~5 minutes  
→ Full incremental update of 50+ symbols in <2 seconds  
→ From 1-minute → 1-year bars in a single pass

Zero-database. Crash-proof. 1M+ candles/sec. Pure Python.
